quickhull is a recursive function used to find the points of the convex hull

Parameters
x,yendpoints of line segment side: the side of xy that we want to find points in
sideCross product value - side of xy

for each point check if it lies on the required side of the line xy

if it does, then find distance of point from xy

if no such point p with max distance exists, add endpoints x and y to Convex Hull

else, recursively find points on the other side of px and py
